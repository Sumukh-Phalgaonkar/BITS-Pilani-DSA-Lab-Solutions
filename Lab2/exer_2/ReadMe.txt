Solution Exercise 2a: FileCopier.c copies the contents of source file into destination file. It detects errors if source file is not detected or if there is a problem in writing to destination file. However if destination file does not exist, it creates one with the name provided. SourceFile.txt contains some dummy text. upon invoking $./FileCopier SouceFile.txt DestinationFile.txt from terminal, it will copy the contents into DestinationFile.txt.

Any other file name can be provided as command line argument.
